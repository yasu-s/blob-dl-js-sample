# 概要

textareaで入力した文字列をファイルで保存するための最低限のサンプルです。

# 実行環境

* Node.js - 10.x
* Yarn - 1.12.x
* lite-server - 2.5.x

# 動作確認

## 1. サンプルのダウンロード

```
git clone git@github.com:yasu-s/blob-dl-js-sample.git
```

## 2. パッケージインストール  

```
cd blob-dl-js-sample
yarn
```

## 3. サンプルの起動  

```
yarn start
```

## 4. 実行結果

http://localhost:3000/
